# Portfolio Website

This is a personal portfolio website built with **React**, **Vite**, and **Tailwind CSS**. It showcases my projects, skills, education, certifications, and experience in a clean and modern design.

## Features

- Responsive and mobile-friendly layout
- Smooth navigation and interactive UI
- Project showcase with details
- Downloadable resume
- Contact form
- Modern design with Tailwind CSS

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Font Awesome](https://fontawesome.com/)

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Start development server:**
	```bash
	npm run dev
	```
3. **Build for production:**
	```bash
	npm run build
	```

## Deployment

This site can be deployed to GitHub Pages using:

```bash
npm run deploy
```
